Aquí va el párrafo 1. Escribe normal, texto plano, sin códigos ni etiquetas. El sistema lo numera solo como NODE_01.

Aquí va el párrafo 2. Cada párrafo se separa del anterior con UNA línea en blanco. Eso es todo el formato que existe.

Aquí va el párrafo 3. La coma y el punto importan: el ADHD MODE corta el texto exactamente por ahí.

Aquí va el párrafo 4. Si quieres una imagen dentro de un párrafo (artes visuales), súbela a la carpeta assets/img y escribe: ![descripción](/assets/img/nombre-archivo.jpg)

Aquí va el párrafo 5. No uses negritas, ni títulos, ni enlaces raros. Palabra pura.

Aquí va el párrafo 6. Recuerda: deben ser EXACTAMENTE 7 párrafos. Si publicas con 6 o con 8, la página muestra una alerta ⚠ visible.

Aquí va el párrafo 7. Borra todos estos textos de ejemplo antes de publicar. El encabezado de arriba (entre los ---) también se edita: title, numero, categoria (manifiesto / palabra / musica / cine / arte / viajes) y autor.
